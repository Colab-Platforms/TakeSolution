# Quick Start: Paste Link Feature

## 🚀 Fastest Way to Add PDFs (GitHub Method)

### 1. Upload to GitHub (One-time setup)
```
1. Go to: https://github.com/YOUR_USERNAME/YOUR_REPO
2. Click: Releases → Create a new release
3. Tag: "investor-pdfs" (or any name)
4. Drag & drop your PDF files
5. Click: Publish release
```

### 2. Get PDF Link
```
1. Right-click on the PDF file
2. Select: "Copy link address"
3. You'll get something like:
   https://github.com/user/repo/releases/download/investor-pdfs/report.pdf
```

### 3. Add to Admin Panel
```
1. Login to admin panel
2. Go to: Investor Management → [Select Tab]
3. Click: "Add New"
4. Fill in: Year, Description, etc.
5. Select: "Paste Link" (radio button)
6. Paste: Your GitHub URL
7. Click: Create
```

## ✅ Done!
Your PDF is now live on the website, hosted on GitHub's CDN for free!

## 🔗 Other Free Options

### Google Drive
1. Upload PDF → Share → "Anyone with link"
2. Copy link
3. Paste in admin panel

### Dropbox
1. Upload PDF → Share → Create link
2. Change `?dl=0` to `?dl=1`
3. Paste in admin panel

---

## 💡 Pro Tips

- Use GitHub for public investor documents (best option)
- Keep filenames descriptive: `Q1-FY26-Results.pdf`
- Test the link before submitting (click preview)
- Don't delete files from external storage after adding
